# ATextKit  

このライブラリは、iOS18以前でもSwiftUIでUITextViewを使い、SwiftUI下でもリッチなテキストの設定ができるようにしたものです  

<span style="font-size:12pt">iOS 17.0+</span>  

## OverView  
このATextKitは、OS18以前のSwiftUIフレームワークでもUIKit並みにリッチなテキスト編集を提供します  
<span style="font-size:10pt">注) SwiftUIでは、Linkの文字属性が非対応のため、それ以外の属性が使えます</span>
